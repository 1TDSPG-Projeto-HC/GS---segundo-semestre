# MindWork — Frontend Moderno

Protótipo frontend do MindWork (bem-estar no trabalho).

## Como rodar

1. Instale dependências:

```bash
npm install
```

2. Rode em desenvolvimento:

```bash
npm run dev
```

Abra http://localhost:5173

## Tecnologias
- React + Vite + TypeScript
- TailwindCSS 3.3.3
- Framer Motion
